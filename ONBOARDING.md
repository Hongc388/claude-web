## Your Setup Checklist

### Codebases
- [ ] claude-web — github.com/hongc388/claude-web

### MCP Servers to Activate
- [ ] ide — Connects Claude Code to your IDE (VS Code or JetBrains) for in-editor context and diagnostics. Install the Claude Code extension from your IDE's marketplace, then restart Claude Code.

### Skills to Know About
- /config — View and edit Claude Code settings (permissions, model, hooks). Use it when you need to adjust what Claude is allowed to do in this repo.
- /usage — Check your Anthropic API usage and token consumption for the current billing period.
- /memory — Open the persistent memory system. Claude uses this to remember your preferences and project context across sessions.
- /init — Bootstrap a new project with a CLAUDE.md and .claude/ directory. Run this first in any repo you want Claude to understand.
- /compact — Summarise and compress the current conversation to free up context window space without losing the thread.
- /tasks — Open the background tasks panel to see what agents are running.
